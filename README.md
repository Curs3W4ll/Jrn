# Jrn

Journalize your working days

## Description

Jrn is a tool to help you know at time what you've done in the day

![Help](/img/help.png)

## Installation

```
git clone https://github.com/Curs3W4ll/jrn.git .jrn
cd .jrn
chmod +x build.sh
./build.sh
```

## Removing

```
cd .jrn
chmod +x remove.sh
./remove.sh
cd ..
rm -rf .jrn
```
